# Support

## Where to Get Help

- Usage questions and troubleshooting:
  - https://github.com/ychampion/codeclaw/issues
- Feature proposals:
  - Open a "Feature request" issue
- Security concerns:
  - Follow `SECURITY.md` and report privately

## What to Include in Support Requests

- command(s) executed
- config context (with secrets removed)
- expected vs actual behavior
- OS + Python version
- relevant logs/errors

## Project Scope Notes

CodeClaw focuses on:

- exporting Claude Code and Codex sessions
- privacy-aware redaction and attestation workflows
- Hugging Face dataset sync
- MCP memory tooling for retrieval over session history
